# 🚀 打造你的专属“隐形飞船”：Xray科学上网傻瓜式教程

## 一、探险前的准备：你需要什么？

在开始建造你的“隐形飞船”之前，我们需要准备一些必要的“工具”和“物资”。别担心，我会一步步指引你！

1.  **一台海外的“云电脑”（VPS）**：
    *   **这是你的飞船核心！** VPS（Virtual Private Server，虚拟专用服务器）就像你租用了一台在海外的电脑。我们将在上面搭建我们的“隐形飞船”系统。
    *   **推荐选择：**
        *   **Vultr**：按小时计费，灵活，新手友好。
        *   **搬瓦工 (BandwagonHost)**：对新手很友好，线路优化好，但价格稍高。
        *   **Contabo/Racknerd/GigsGigsCloud等**：价格更实惠，但可能对新手来说配置稍微复杂一点点。
        *   **选择小贴士：** 尽量选择机房在美西、日本、新加坡、韩国等距离你较近的地区，能获得更好的速度。操作系统选择 **Ubuntu 20.04 LTS 或 Debian 11**。

2.  **一个“飞船口令”（SSH 工具）**：
    *   SSH工具让你能够远程登录并控制你的VPS。
    *   **Windows 用户：** 推荐使用 **PuTTY** 或更现代的 **Termius**、**Xshell**。
    *   **macOS/Linux 用户：** 系统自带的 **终端 (Terminal)** 即可。
    *   **手机用户：** 推荐 **Termius** (iOS/Android) 或 **JuiceSSH** (Android)。

3.  **一个“飞船导航系统”（域名，可选但强烈推荐）**：
    *   域名就像你飞船的独一无二的“名字”（例如 `your-awesome-ship.com`）。它能让你的飞船更稳定，更不易被发现。
    *   **获得方式：**
        *   **免费域名：** 可以尝试 Freenom（`.tk`, `.ml` 等），但稳定性不佳。
        *   **付费域名：** 推荐 Namecheap、Cloudflare Registrar、腾讯云/阿里云国际版等，价格一年几十块钱，非常值得投资。
    *   **重要！** 如果你有域名，请务必将其解析到你的VPS的IP地址上。我们后面会用到Cloudflare来管理DNS。

4.  **一颗探索未知的好奇心和耐心！**

## 二、建造飞船第一步：租用你的“云电脑”（VPS）

以 Vultr 为例，演示如何购买一台 VPS。其他服务商步骤类似。

1.  **注册并登录 Vultr 官网**：访问 [https://www.vultr.com/](https://www.vultr.com/)，注册一个账号。新用户通常会有一些免费额度。
2.  **充值（Funding）**：根据你的需求充值。通常一个月最低配置的VPS在5-10美元左右。支持支付宝、微信支付、信用卡等。
3.  **部署一台新服务器 (Deploy New Server)**：
    *   点击左侧菜单的 `Products` -> `Instances`，然后点击右侧的 `+` 号部署新服务器。
    *   **选择服务器类型 (Choose Server Type)**：`Cloud Compute` (最常用)。
    *   **选择CPU架构 (CPU & Storage Technology)**：`AMD High Performance` 或 `Intel High Performance`。
    *   **选择服务器位置 (Server Location)**：选择距离你较近且网络质量好的地区，如 `Tokyo`（东京）、`Singapore`（新加坡）、`Los Angeles`（洛杉矶）。
    *   **选择操作系统 (Server Operating System)**：
        *   `Ubuntu` -> `Ubuntu 20.04 LTS` 或 `Ubuntu 22.04 LTS`
        *   `Debian` -> `Debian 11`
        *   **注意：** 教程将以 Ubuntu 为例，如果你选择其他系统，部分命令可能略有不同。
    *   **选择服务器配置 (Server Size)**：选择最便宜的配置（通常是 1C1G 或 2C1G），对于个人使用足够了。
    *   **其他设置 (Optional Features)**：保持默认即可。
    *   **部署 (Deploy Now)**：点击部署，等待几分钟，你的VPS就会创建成功。

4.  **获取VPS信息**：
    *   部署成功后，你会在 `Instances` 页面看到你的VPS，状态变为 `Running`。
    *   点击你的VPS实例，你会看到它的 **IP Address**、**Username** (通常是 `root`) 和 **Password**。这些信息非常重要，务必保存好！

## 三、建造飞船第二步：连接你的“云电脑”

现在，我们有了VPS的IP、用户名和密码，就像有了飞船的地址和开门密码。接下来，我们用SSH工具连接它。

**以 Windows 下 PuTTY 为例：**

1.  **下载 PuTTY**：访问 [https://www.chiark.greenend.org.uk/~sgtatham/putty/latest.html](https://www.chiark.greenend.org.uk/~sgtatham/putty/latest.html) 下载最新版。
2.  **打开 PuTTY**：
    *   在 `Host Name (or IP address)` 处输入你的VPS的 **IP地址**。
    *   `Port` 保持默认 `22`。
    *   `Connection type` 保持 `SSH`。
    *   点击 `Open`。
3.  **登录**：
    *   首次连接可能会弹出一个安全警告，点击 `Accept`。
    *   `login as:` 输入用户名 `root`，回车。
    *   `password:` 输入你的VPS密码（输入时不会显示字符，这是正常的），回车。
    *   如果一切顺利，你将看到一个命令行界面，表示你已成功登录VPS！

**以 macOS/Linux 下终端 (Terminal) 为例：**

1.  **打开终端**。
2.  输入命令：`ssh root@你的VPS的IP地址`，回车。
    *   例如：`ssh root@192.0.2.1`
3.  首次连接会询问是否继续连接，输入 `yes`，回车。
4.  `password:` 输入你的VPS密码，回车。
5.  成功登录后，你将看到命令行界面。

## 四、建造飞船第三步：安装“隐形飞船系统”（Xray 一键脚本）

这一步是核心！我们将使用一个“一键脚本”来自动化安装Xray及其所需的所有组件，这大大降低了技术门槛。就像给飞船安装了自动驾驶系统！

**⚠️ 重要提示：**
*   请仔细复制粘贴命令，不要手动输入，以免出错。
*   每次粘贴命令后，都按回车键执行。

1.  **更新系统包**：
    刚进入VPS，最好先更新一下系统。
    ```bash
    sudo apt update && sudo apt upgrade -y
    ```
    （如果你使用的是 `Debian` 或 `Ubuntu` 系统，这个命令是通用的。如果遇到提示，一路按 `Y` 或回车。）

2.  **下载并运行 Xray 一键安装脚本**：
    市面上有许多优秀的Xray一键安装脚本，这里我们使用一个功能完善且维护较好的脚本。它将帮你完成Xray的安装、配置，甚至可以帮你配置SSL证书（如果你有域名的话）。

    ```bash
    bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)
    ```
    这个脚本是一个流行的面板安装脚本，它会先安装 `3x-ui` 面板，然后通过面板来方便地管理Xray配置。

    **脚本执行过程：**
    *   脚本会下载并安装 `3x-ui` 面板。
    *   安装完成后，它会显示 `3x-ui` 的登录地址、端口、用户名和密码。请务必保存好这些信息！例如：
        ```
        面板访问地址: http://你的VPS的IP:端口/
        用户名: admin
        密码: admin
        ```
        （端口通常是随机生成的，例如 `20000-65535` 之间的某个端口）
    *   如果面板安装失败，请检查你的系统是否为 Ubuntu/Debian，并尝试重新执行脚本。

3.  **登录 `3x-ui` 面板并添加配置**：

    *   在你的浏览器中，输入上一步中获取到的 **面板访问地址**（例如：`http://你的VPS的IP:随机端口/`）。
    *   输入 **用户名 `admin`** 和 **密码 `admin`**（如果脚本没有提示修改，初始就是 `admin`）。首次登录后建议修改密码！
    *   登录后，你将看到 `3x-ui` 的管理界面。
    *   点击左侧菜单的 `入站列表`，然后点击 `添加入站`。

    *   **配置“隐形飞船”协议 (Xray Inbound Settings)**：
        我们选择一种既安全又隐蔽的配置：**VLESS + TLS + WebSocket + Web伪装**。
        *   **备注**：给自己起个好记的名字，例如 `MyXrayServer`。
        *   **协议**：选择 `VLESS`。
        *   **端口**：输入 `443` (这是HTTPS的默认端口，非常隐蔽)。
        *   **UUID**：点击旁边的 `生成` 按钮，生成一个随机的UUID。这是你飞船的独特识别码。
        *   **传输协议 (Transmission)**：选择 `ws` (WebSocket)。
        *   **底层安全 (TLS)**：选择 `tls`。
        *   **流控 (Flow)**：留空或者选择 `xtls-rprx-vision` (如果你VPS性能好且客户端支持)。
        *   **域名**：如果你有域名并已经解析到此VPS IP，在此处填入你的域名（例如 `your-awesome-ship.com`）。**这一步非常重要，没有域名就无法开启TLS和伪装。**
            *   **如果没有域名怎么办？** 你可以先不填域名，暂时只用 `VLESS + WS`，但这种方式不如带域名的安全和稳定。强烈建议你去注册一个域名。
        *   **公钥**：留空。
        *   **私钥**：留空。
        *   **回落**：
            *   **默认回落 (Default Fallback)**：选择 `重定向到网站`。
            *   **回落地址**：`https://www.bing.com` 或 `https://www.baidu.com` (任意一个真实网站)。
            *   **回落路径**：留空。
        *   **SNI**：留空。
        *   **ALPN**：留空。
        *   **开启TLS (Enable TLS)**：勾选。
        *   **申请SSL证书 (Request SSL Certificate)**：勾选。系统会尝试使用 **acme.sh** 自动申请并续期证书。
        *   **伪装域名 (Fake Domain)**：如果你有域名，这里可以填一个常见的网站域名，例如 `www.apple.com` 或 `www.microsoft.com`。这会进一步增加伪装效果。
        *   **路径 (Path)**：可以自定义一个路径，例如 `/ray`。记住这个路径，客户端配置时会用到。
        *   **伪装类型 (Fake Type)**：`none` (保持默认)。
        *   **端口重用 (Reuse Port)**：勾选。
        *   **额外Id (AlterId)**：`0` (VLESS协议通常是0)。
        *   **系统用户 (System User)**：`nobody`。

    *   **点击 `创建`**。
    *   等待片刻，系统会尝试申请SSL证书。如果申请成功，你的Xray配置就搭建好了！

    *   **如果SSL证书申请失败怎么办？**
        *   确保你的域名已经正确解析到你的VPS的IP地址。
        *   确保你的VPS的443端口没有被其他程序占用。
        *   尝试在 `3x-ui` 面板中手动更新 acme.sh。
        *   如果实在不行，可以考虑先不使用域名，或者换一个脚本或方案。

## 五、建造飞船第四步：配置你的“乘客端”（客户端连接）

现在，飞船核心系统已经搭建完毕，我们需要在你的设备上安装“乘客端”软件，让它能连接上你的飞船。

在 `3x-ui` 面板中，你的入站配置旁边会有一个 **二维码图标** 或 **配置文件链接**。点击它，就可以看到你的 **VLESS配置链接 (vmess:// 或 vless:// 开头)** 和 **二维码**。
请复制这个链接或扫描二维码，用于你的客户端配置。

### 1. Windows 客户端（推荐：V2RayN 或 Qv2ray）

以 **V2RayN** 为例：

1.  **下载 V2RayN**：访问其GitHub页面下载最新版本：[https://github.com/2dust/v2rayN/releases](https://github.com/2dust/v2rayN/releases)。下载 `V2RayN-Core.zip` 解压到任意目录。
2.  **运行 V2RayN**：双击 `v2rayN.exe`。它通常会最小化到系统托盘。
3.  **导入配置**：
    *   右键点击系统托盘中的 V2RayN 图标，选择 `从剪贴板导入批量URL` (如果你复制了vless链接)。
    *   或者选择 `扫描屏幕上的二维码` (如果你在电脑屏幕上显示了二维码)。
    *   导入成功后，你会看到你的服务器配置出现在列表中。
4.  **选择服务器**：右键点击你导入的服务器，选择 `设为活动服务器`。
5.  **开启系统代理**：
    *   右键点击系统托盘中的 V2RayN 图标，选择 `系统代理` -> `自动配置系统代理`。
    *   或者选择 `系统代理` -> `PAC模式` 或 `全局模式`。`PAC模式` 智能分流，`全局模式` 所有流量都走代理。

### 2. Android 客户端（推荐：V2RayNG 或 Kitsunebi）

以 **V2RayNG** 为例：

1.  **下载 V2RayNG**：在 Google Play 商店搜索 `V2RayNG` 安装，或者从其GitHub页面下载 `apk` 文件安装：[https://github.com/2dust/v2rayNG/releases](https://github.com/2dust/v2rayNG/releases)。
2.  **导入配置**：
    *   打开 V2RayNG。点击右上角的 `+` 号。
    *   选择 `从剪贴板导入` (如果你复制了vless链接)。
    *   选择 `扫描二维码` (扫描电脑屏幕上的二维码)。
3.  **连接**：
    *   导入成功后，点击 V2RayNG 主界面右下角的圆形按钮（通常是V图标或纸飞机图标）即可连接。
    *   首次连接可能会提示VPN权限，点击 `允许`。

### 3. iOS 客户端（推荐：Shadowrocket 小火箭 或 Quantumult X 等）

这些软件在App Store需要付费购买，但通常功能强大且稳定。

以 **Shadowrocket (小火箭)** 为例：

1.  **购买并下载 Shadowrocket**：在 App Store 搜索 `Shadowrocket` 购买并下载。
2.  **导入配置**：
    *   打开 Shadowrocket。
    *   点击左上角的 `+` 号。
    *   选择 `Scan QR Code` (扫描电脑屏幕上的二维码)。
    *   或者选择 `Add Server` -> `Type` 选择 `VLESS`，然后手动填写服务器信息（地址、端口、UUID、传输协议、TLS、路径等）。如果你复制了VLESS链接，可以直接选择 `From Clipboard`。
3.  **连接**：
    *   导入成功后，选择你刚刚添加的服务器。
    *   在主界面顶部开启 `Connect` 开关即可。
    *   首次连接会提示VPN权限，点击 `Allow`。

## 六、测试你的“隐形飞船”

当你配置好客户端并连接后，就可以测试你的“飞船”是否正常工作了！

1.  **访问 Google、YouTube**：尝试访问一些在国内无法直接访问的网站，如 [https://www.google.com/](https://www.google.com/) 或 [https://www.youtube.com/](https://www.youtube.com/)。
2.  **测速**：
    *   访问 [https://fast.com/](https://fast.com/) 或 [https://www.speedtest.net/](https://www.speedtest.net/) 测试你的网速。

## 七、常见问题与“踩坑”经验

1.  **速度慢或不稳定**：
    *   **更换VPS节点**：尝试更换其他地区的VPS，如日本、韩国、美西等。
    *   **检查VPS流量**：确保你的VPS流量没有用完。
    *   **更换传输协议或伪装方式**：在 `3x-ui` 面板中尝试其他协议（如 Trojan、XTLS 等），但需要客户端支持。
    *   **检查网络环境**：有时候是本地网络问题，尝试切换WIFI或数据流量。

2.  **无法连接**：
    *   **检查IP是否被墙**：这是最常见的问题。你的VPS的IP地址可能被“墙”了。
        *   登录VPS，执行 `ping google.com`，如果ping不通，很可能IP被墙。
        *   **解决方案**：
            *   在VPS服务商处更换IP地址（可能需要额外费用或销毁重建）。
            *   更换新的VPS。
    *   **检查端口是否开放**：确保你VPS的防火墙允许443端口（或你设置的端口）的流量通过。Vultr通常默认开放所有端口，但某些服务商或系统自带防火墙可能会拦截。
        *   在VPS中执行 `sudo ufw status` (如果安装了ufw防火墙)。确保443端口是开放的。
    *   **检查配置是否正确**：仔细核对客户端的配置与 `3x-ui` 面板中导出的信息是否完全一致，尤其是 **UUID**、**域名**、**路径**、**端口**、**底层安全**、**传输协议**。
    *   **证书问题**：如果使用TLS，确保SSL证书申请成功且在有效期内。

3.  **VPS被封号**：
    *   极少数情况下，如果你的流量异常大或VPS服务商检测到违规行为，可能会封号。
    *   **解决方案**：更换服务商。

4.  **如何续费/管理VPS**：
    *   登录你的VPS服务商官网，在账单或实例管理页面进行续费。
    *   建议开启自动续费，以免服务到期被暂停。
