# 07 导入导出使用教程  
  
云端书签支持多种方式导入导出书签，方便你从其他地方迁移过来，或者备份数据。  
  
---  
  
## 导入书签  
  
### 从浏览器书签栏直接导入  
  
这是最方便的，如果你之前把书签存在浏览器原生书签栏，可以直接导入：  
  
1. 打开设置页面。  
2. 找到「导入导出」区域。  
3. 点击「从浏览器书签栏导入」。  
4. 在弹出框选择要导入到哪个场景。  
5. 点击确认，插件会自动读取浏览器书签。  
6. 导入完成会告诉你新增了多少个书签。  
  
> 注意：有些移动浏览器不支持书签 API，这种方法可能用不了，你可以用 HTML 文件导入。  
  
### 从 HTML 文件导入  
  
如果你从其他浏览器或其他书签工具导出了 HTML 文件，可以用这个方法：  
  
1. 准备好 HTML 文件（一般浏览器书签导出都是这个格式）。  
2. 在设置页点击「从文件导入」。  
3. 选择你的 `.html` 文件。  
4. 选择要导入到哪个场景。  
5. 点击确认，插件会解析文件里的书签和文件夹结构。  
6. 导入完成提示结果。  
  
### 从 JSON 文件导入  
  
如果你之前从云端书签导出过 JSON 备份，可以用这个方法恢复：  
  
1. 在设置页点击「从文件导入」。  
2. 选择 `.json` 文件。  
3. 选择目标场景。  
4. 确认导入。

## 导出书签  
  
### 导出为 JSON 文件（推荐用于备份）  
  
JSON 格式保留所有信息：书签、文件夹、备注、标签、收藏状态、排序都完整保留。建议定期导出备份。  
  
导出方法：  
- **方法一**：在设置页 → 导入导出区域 → 点击「导出为 JSON」。  
- **方法二**：在书签管理页右上角 → 点击「导出」→ 选择「导出为 JSON」。  
  
导出的文件可以保存到本地，万一出问题可以用这个恢复。  
  
### 导出为 HTML 文件（用于迁移到浏览器）  
  
HTML 格式是浏览器通用格式，可以导入到其他浏览器或其他书签工具：  
  
导出方法：  
- **方法一**：在设置页 → 导入导出区域 → 点击「导出为 HTML」。  
- **方法二**：在书签管理页右上角 → 点击「导出」→ 选择「导出为 HTML」。  
  
导出后，你可以打开浏览器书签管理器，导入这个 HTML 文件。  
  
> 注意：不管哪种导出方式，都只导出**当前场景**的书签，其他场景不会导出。

## WebDAV 配置导入导出  
  
如果你要在多台设备上使用，不用每次手动输入 WebDAV 配置，可以一键导入导出：  
  
### 导出配置  
1. 在已经配置好的设备上，打开设置页。  
2. 找到「WebDAV 配置工具」区域。  
3. 点击「导出到剪贴板」。  
4. 你的配置就复制到剪贴板了，格式是三行：  
   ```  
   第一行：WebDAV 地址  
   第二行：用户名   第三行：密码   ```  
### 导入配置  
1. 在新设备上打开设置页。  
2. 点击「导入配置」。  
3. 在弹出框粘贴你刚才复制的三行配置。  
4. 点击「导入」。  
5. 地址、用户名、密码会自动填到表单里。  
6. 你检查一下没错，点击「保存配置」就好了。

## 导入规则说明  
  
- **自动去重**：同一个网址在同一个场景里已经有了，不会重复导入。  
- **保留文件夹结构**：原来的文件夹层级会保留。  
- **清理空文件夹**：导入完会自动删掉没有用到的空文件夹。  
- **你选择目标场景**：所有导入的书签都会放到你选的场景里，不会影响其他场景。

## 常见问题  
  
### Q: 导入会覆盖我现有的书签吗？  
A: 不会。导入是合并进去，只会新增，不会删除你现有的。只有网址完全一样才会跳过，不会覆盖。  
  
### Q: 支持什么格式的 HTML 文件？  
A: 支持所有浏览器标准导出的 HTML 书签格式，Chrome、Edge、Firefox 导出的都能用。  
  
### Q: 我想把一个场景的书签复制到另一个场景怎么办？  
A: 先从原场景导出 JSON，然后在新场景导入，就复制过去了。
