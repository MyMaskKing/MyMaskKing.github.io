## 💡 什么是“科学上网”？为什么我们需要它？

“科学上网”，简单来说，就是通过一些技术手段，让你的网络流量绕过本地的网络限制，连接到境外的服务器，再由境外的服务器访问互联网。这样，你就好像拥有了一张“国际通行证”，可以自由访问全球范围内的网站和应用了。

**为什么我们需要它？**

*   **获取全球资讯：** 访问被本地限制的国际新闻、学术资料、技术博客等。
*   **娱乐无界：** 观看Netflix、YouTube等平台上的独家内容，畅玩国际服游戏。
*   **工作学习：** 使用Google、GitHub等国际协作和开发工具，与世界同步。
*   **隐私保护：** 有效隐藏你的真实IP地址，加密你的网络数据，提升上网安全性。

## 🛠️ 核心工具准备：打造你的“高速公路”基地

搭建你的专属“高速公路”需要几个关键的“建筑材料”和“工具”。别怕，它们都非常容易获取和使用！

### 1. **一台“海外服务器” (VPS)**

VPS (Virtual Private Server)，直译过来就是“虚拟私人服务器”。你可以把它想象成你在国外租了一间小公寓，这间公寓有独立的网络线路，并且24小时不打烊。你的所有网络请求都会先通过这间公寓转发出去。

*   **推荐选择：** 初学者建议选择国外主流的VPS提供商，如：
    *   **Vultr:** 按小时计费，配置灵活，全球节点多。
    *   **DigitalOcean:** 界面友好，文档丰富，适合新手。
    *   **搬瓦工 (BandwagonHost):** 线路质量好，但价格相对较高。
    *   **Contabo:** 价格便宜，配置高，但部分节点速度可能一般。
*   **操作系统 (OS)：** 强烈推荐选择 **Ubuntu 22.04 LTS** 或 **Debian 11/12**。它们稳定、易用，并且有大量的教程支持。

### 2. **一个“导航地址” (域名，可选但强烈推荐)**

域名（Domain Name）就是你访问网站时输入的网址，比如 `google.com`。在这里，它将作为你VPS的“门牌号”，让你的科学上网服务看起来更像一个普通的网站，从而增加稳定性和安全性。

*   **获取方式：** 可以在Namesilo、Cloudflare等平台注册购买。注册一个便宜的后缀（如`.xyz`, `.top`）即可。
*   **作用：** 配合TLS加密，能有效伪装你的流量，降低被识别的风险。

### 3. **一个“施工队长” (SSH客户端)**

SSH（Secure Shell）是一种加密的网络协议，它能让你通过命令行远程连接并操作你的VPS。

*   **Windows 用户：** 推荐使用 **PuTTY** 或 **Termius** (功能更强大，颜值更高)。
*   **macOS / Linux 用户：** 直接使用系统自带的“终端 (Terminal)”即可。
*   **手机用户：** 推荐 **Termius** (Android/iOS) 或 **JuiceSSH** (Android)。

## 🚧 搭建步骤详解：一步步盖起你的“高速公路”

下面我们将正式进入搭建阶段！我将以最常用、最稳定的 **Xray (VLESS+XTLS)** 协议为例，配合一个非常傻瓜化的一键安装脚本，让你轻松搞定服务端配置。

### 第一步：购买并初始化你的VPS 🏠

1.  **选择并购买VPS：**
    *   访问你选择的VPS提供商网站（例如Vultr）。
    *   注册账号，绑定支付方式（信用卡或PayPal）。
    *   选择一个**服务器位置 (Server Location)**，建议选择离你地理位置较近、网络线路较好的地区，如：日本东京、韩国首尔、美国西海岸（洛杉矶、西雅图）、新加坡等。
    *   选择一个**操作系统 (Operating System)**，**务必选择 Ubuntu 22.04 LTS x64**。
    *   选择合适的配置套餐。对于个人使用，最低配（1核CPU，1GB内存，25GB硬盘）通常足够。
    *   确认并创建服务器。等待几分钟，你的VPS就会创建成功，并显示出IP地址、用户名（通常是`root`）和密码。
2.  **记录VPS信息：** 把VPS的 **IP地址**、**用户名 (root)** 和 **密码** 复制并保存好，这是你连接VPS的“钥匙”。

### 第二步：配置你的“导航地址” (域名解析) 🗺️ (推荐！)

如果你购买了域名，这一步能让你的服务更稳定、更安全。如果没有，可以跳过，但安全性会稍逊。

1.  **登录你的域名注册商（如Namesilo/Cloudflare）的控制面板。**
2.  **找到DNS管理界面。**
3.  **添加一条A记录：**
    *   **类型 (Type)：** `A`
    *   **主机记录 (Host/Name)：** 填写一个你想要的二级域名，例如 `vpn` 或 `proxy`。如果你想直接用主域名，这里填 `@`。
    *   **记录值 (Value)：** 填写你**VPS的IP地址**。
    *   **TTL (Time To Live)：** 建议设置为 `自动` 或 `600` 秒。
4.  **保存设置。** DNS解析通常需要几分钟到几小时才能全球生效。你可以使用 [https://www.whatsmydns.net/](https://www.whatsmydns.net/) 查询你的域名解析是否生效。当查询到你的域名指向了VPS的IP地址时，就表示成功了。

### 第三步：连接你的VPS并运行一键安装脚本 🖥️

这一步是核心！我们将使用一个功能强大且简单易用的一键安装脚本，来自动配置Xray服务器。这里我们选择 **Wulabing 的 Xray 一键脚本**，它对 VLESS+XTLS 的支持非常完善。

1.  **打开你的SSH客户端** (PuTTY/Termius/终端)。
2.  **连接VPS：**
    *   **IP地址：** 输入你VPS的IP地址。
    *   **端口：** 默认是 `22`。
    *   **用户名：** `root`
    *   **密码：** 输入你VPS的密码 (输入时屏幕上不会显示字符，输完按回车)。
    *   如果连接成功，你会看到一个类似 `root@your-vps-name:~#` 的命令行提示符。恭喜你，你已经进入你的“海外公寓”了！
3.  **更新系统 (可选但推荐)：**
    *   在SSH客户端中输入以下命令，更新系统软件包：
        ```bash
        sudo apt update && sudo apt upgrade -y
        ```
        等待更新完成。
4.  **运行Xray一键安装脚本：**
    *   输入以下命令，下载并运行Wulabing的Xray一键安装脚本：
        ```bash
        wget -N --no-check-certificate -q "https://raw.githubusercontent.com/wulabing/Xray_onekey/main/install.sh" && chmod +x install.sh && bash install.sh
        ```
    *   脚本会开始运行，并出现一个菜单。按照以下步骤操作：
        *   **选择安装Xray (选项1)。**
        *   **选择VLESS+XTLS (选项2)。**
        *   **是否申请域名证书 (选项y)。** 如果你配置了域名，这里选 `y`。
        *   **请输入你的域名：** 输入你在第二步解析好的域名（例如 `vpn.yourdomain.com`）。
        *   **是否开启BBR加速 (选项y)。** 开启BBR能显著提高网络速度。
        *   **输入伪装网站：** 可以随便输入一个合法网站（如 `https://www.apple.com`），用于伪装流量。
        *   **回车确认。**
    *   脚本会自动安装Xray，申请TLS证书，并配置服务。这个过程可能需要几分钟。
5.  **记录配置信息：**
    *   脚本运行成功后，会显示你的Xray配置信息，包括：
        *   **IP地址**
        *   **端口 (Port)**
        *   **UUID (用户ID)**
        *   **AlterId (通常为0)**
        *   **流控 (Flow: xtls-rprx-direct)**
        *   **TLS (已开启)**
        *   **伪装域名 (Fake Domain)**
        *   **客户端配置URL (vmess/vless)** 和 **二维码 (QR Code)**
    *   **将这些信息全部复制保存好，尤其是URL和QR Code！** 这是你客户端连接的关键。

### 第四步：配置你的“乘客” (客户端设置) 📱💻

现在服务器已经搭建好了，我们需要在你的设备上安装客户端，并用刚刚获取的配置信息连接到服务器。

#### **A. Windows 用户：使用 V2RayN**

1.  **下载 V2RayN：** 访问 [V2RayN GitHub Releases](https://github.com/2dust/v2rayN/releases) 下载最新版的压缩包（选择 `v2rayN-Core.zip`）。
2.  **解压并运行：** 将下载的压缩包解压到一个你喜欢的目录，运行 `v2rayN.exe`。
3.  **导入配置：**
    *   在任务栏托盘找到V2RayN图标（一个小飞机），右键点击。
    *   选择 `从剪贴板导入批量URL` (如果你复制了URL) 或 `扫描屏幕二维码` (如果你在电脑上显示了二维码)。
    *   你也可以手动添加：`服务器` -> `添加 VLESS 服务器`。然后将服务端输出的UUID、地址、端口、流控（xtls-rprx-direct）、TLS等信息一一填写。
4.  **启用系统代理：**
    *   在V2RayN主界面，选择你导入的服务器。
    *   在任务栏托盘图标右键，选择 `系统代理` -> `自动配置系统代理`。
5.  **开始连接：**
    *   确保你的服务器已选中，点击V2RayN界面的“启动”按钮或在托盘图标右键选择“启动服务”。
6.  **测试连接：** 打开浏览器，访问 [https://www.google.com](https://www.google.com) 或 [https://ip.gs](https://ip.gs) 查看你的IP地址是否已变为VPS的IP。

#### **B. Android 用户：使用 V2RayNG**

1.  **下载 V2RayNG：**
    *   在Google Play商店搜索 `V2RayNG` 并安装。
    *   如果没有Google Play，可以在 [V2RayNG GitHub Releases](https://github.com/2dust/v2rayNG/releases) 下载最新版 `apk` 文件安装。
2.  **导入配置：**
    *   打开V2RayNG。
    *   点击右上角的 `+` 号。
    *   选择 `扫描二维码` (扫描你VPS终端上显示的二维码) 或 `从剪贴板导入` (如果你复制了URL)。
    *   你也可以选择 `手动输入 VLESS`，然后填入相应信息。
3.  **连接：**
    *   在V2RayNG主界面，选择你导入的服务器。
    *   点击右下角的圆形连接按钮 (通常是飞机图标)。
    *   首次连接会提示VPN权限请求，点击“允许”。
4.  **测试连接：** 打开浏览器，访问 [https://www.google.com](https://www.google.com) 或 [https://ip.gs](https://ip.gs) 确认。

#### **C. iOS 用户：使用 Shadowrocket 或 Quantumult X (付费)**

1.  **购买并下载客户端：** 在App Store搜索 `Shadowrocket` (俗称小火箭) 或 `Quantumult X`。这些都是付费应用，需要使用非中国大陆区的Apple ID购买。
2.  **导入配置：**
    *   **Shadowrocket:**
        *   打开Shadowrocket，点击右上角的 `+` 号。
        *   选择 `Scan QR Code` (扫描二维码) 或 `Add URL` (复制链接)。
    *   **Quantumult X:**
        *   打开Quantumult X，点击右下角的小风车图标进入配置。
        *   点击 `节点` -> `添加` -> `从剪贴板导入` 或 `扫描二维码`。
3.  **启动连接：**
    *   **Shadowrocket:** 选择导入的节点，打开顶部的连接开关。
    *   **Quantumult X:** 启用右上角的开关。
4.  **测试连接：** 浏览器访问 [https://www.google.com](https://www.google.com) 确认。

## ⚠️ 重要提示和踩坑经验：让你的“高速公路”更稳固

*   **定期更新：** 操作系统和Xray程序都需要定期更新，以修复漏洞和提升性能。对于Xray，可以再次运行一键脚本，选择更新选项。
*   **IP被墙？** 如果你的VPS IP突然无法连接，可能是IP被墙了。联系VPS提供商更换IP地址，或者重新购买一台VPS。
*   **流量伪装：** 强烈建议使用域名并开启TLS，这能大大提高你的服务稳定性，降低被识别和封锁的风险。
*   **避免滥用：** 请勿使用科学上网进行违法违规活动，遵守当地法律法规。
*   **备份配置：** 将你的Xray配置信息妥善保存，以防需要重新配置或在其他设备上使用。
*   **选择合适的VPS：** 如果你发现速度不理想，可能需要尝试更换VPS提供商、更换节点位置，或者升级VPS配置。一些热门区域的VPS（如香港、日本）速度通常较好，但价格也高。
*   **BBR加速：** 脚本中默认开启了BBR加速，它能有效提高TCP连接的吞吐量和速度，请确保开启。
*   **防火墙：** VPS的默认防火墙通常允许必要的端口（如SSH的22，以及Xray使用的端口）。如果遇到连接问题，检查一下VPS提供商的防火墙设置，确保相关端口已开放。
